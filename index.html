What is Responsive Web Design?

Responsive Web Design makes the web page look good on all devices by the use of Grids and Media Queries in html and CSS.

Relevant Syntax:

HTML: Head tag: meta name="viewport" content="width=device-width,
          initial-scale=1.0"
CSS: Responsive Grid-view: * {box-sizing: border-box;} &
       Break-points: @media only screen and (max/min-width: __px) { ... ; }

What?

RWD resizes the content to fit the screen of different device sizes.

Where?

HTML & CSS, no programming languages.

How?

Use meta tag in head-section & viewport element, Grid-view and Media Queries.

Resize the browser window & test how the content respond to the resizing.
